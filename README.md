# Timebox Planner

일론 머스크식 타임박스 데일리 플래너 데스크톱 앱.

## 다운로드

👉 **[최신 버전 다운로드 (TimeboxPlanner.exe)](../../releases/latest/download/TimeboxPlanner.exe)**

설치 없이 exe 파일 하나로 실행됩니다 (Windows 7 이상, 32/64비트 모두 지원).

> 처음 실행 시 Windows SmartScreen 경고가 뜨면 **"추가 정보" → "실행"** 을 눌러주세요.

## 주요 기능

- **월간 달력**: 날짜별 메모, 날짜 클릭 시 데일리 플래너로 이동
- **데일리 플래너**: 우선순위 3가지(체크박스) + 브레인스토밍 + 타임박스
- **타임박스**: 드래그로 일정 생성/이동/크기 조절 (10분 단위), 더블클릭으로 편집, 완료 체크
- **서식**: 굵게(Ctrl+B)·밑줄(Ctrl+U)·기울임(Ctrl+I)·취소선·형광펜 6색·글자 크기
- **팝업 모드**: 항상 위 표시, 불투명도 30~100% 조절
- 모든 기록은 날짜별로 자동 저장 (PC 로컬 저장)

## 개발

```bash
npm install
npm start          # 개발 실행
npm run dist       # 배포용 exe 빌드
```
